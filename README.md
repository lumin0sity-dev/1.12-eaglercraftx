# ABOUT

This project is a version of eaglercraftX based on Minecraft 1.12, specifically optimized for performance. It is legally licensed under the MIT License.

## Key Features

* **Version 1.12:** Built upon the core features of Minecraft version 1.12.
* **Performance Focused:** Engineered for enhanced performance and efficiency.
* **Modern Item Textures:** Runtime texture injection system for 1.13-1.21.1 items (160+ items from newer versions)

## Runtime Texture Injection System

This version includes a runtime texture injection system that allows displaying correct textures for items from Minecraft 1.13-1.21.1 when connecting to newer servers. See [TEXTURE_PACK_GUIDE.md](TEXTURE_PACK_GUIDE.md) for details.

**Supported items:**
- 1.13 (Update Aquatic): Coral, kelp, trident, turtle items, fish variants
- 1.14 (Village & Pillage): Crossbow, bamboo, lantern, new flowers, utility blocks
- 1.15 (Buzzy Bees): Honey items, beehive
- 1.16 (Nether Update): Netherite gear, crimson/warped wood, soul items
- 1.17 (Caves & Cliffs): Copper, amethyst, candles, deepslate, spyglass
- 1.18-1.19: Mangrove wood, frog items, sculk, recovery compass
- 1.20 (Trails & Tales): Cherry/bamboo wood, pottery sherds, armor trims
- 1.21 (Tricky Trials): Mace, wind charge, copper doors, tuff variants
